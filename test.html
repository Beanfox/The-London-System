The London System

The london system includes the following moves; d4, Bf4, Nf3

The ideal setup for white while using the London System is in the typical order of

d4, Bf4, Nf3, c5, e6, Bd3, Nbd2

Overall, the London System is a great defensive opening for white that is great to allow for a seemless entrance into the endgame

Have fun playing!
